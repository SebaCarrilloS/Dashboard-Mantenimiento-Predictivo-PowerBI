# Dashboard de Mantenimiento Predictivo y Gestión de Activos

Dashboard desarrollado en Power BI para monitorear condición, riesgo, vida útil remanente y priorización de activos físicos.

## Objetivo
Construir una herramienta de análisis para apoyar decisiones de mantenimiento, priorización de intervención y seguimiento del estado de activos críticos.

## Páginas del dashboard
- Resumen Ejecutivo
- Riesgo y Priorización
- Degradación de Activos
- RUL de Activos
- Detalle de Activo

## Indicadores principales
- Total de activos
- Activos críticos
- Health Index
- Risk Score
- RUL actual y promedio
- Nivel de riesgo
- Acción recomendada

## Funcionalidades
- Navegación lateral entre páginas
- Segmentadores sincronizados
- Botón para limpiar filtros
- Comparación del activo vs promedio de familia
- Tabla de últimos registros operacionales
- Página de diagnóstico individual por activo

## Herramientas utilizadas
- Power BI
- Power Query
- DAX
- Modelado relacional
- Visualización ejecutiva de KPIs
